# LUCID documentation archive

This folder holds **historical or export-only** copies of LUCID documents.

## `LUCID_contracts_master_bundle_v1.0.1_ARCHIVED.md`

This file is a **frozen concatenation** of the v1.0.1 contract set as it existed before M00. It is **not** the canonical editable contract source.

**Canonical source of truth for contracts:** individual files under [`docs/contracts/`](../contracts/).

Do **not** manually dual-maintain the master bundle alongside the individual contracts. If you need an updated export bundle, regenerate it from the canonical files in a controlled process and archive it here with a new version label.
